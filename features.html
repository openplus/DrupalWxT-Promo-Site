Accessible

Accessible

Meets TBS accessibility standards

Provides turnkey WCAG 2.0 Level AA compliance, including user content filters and restrictions to prevent accessibility breakage. ( Passes using TBS endorsed CSE validator).

WxT Themes

Themes

Instant TBS WET-BOEW Themes

Includes a GoC Public and Intranet WxT 3, 3.1 and the newer Bootstrap WxT 4 branch. As well as a plain (non-goc) WxT theme.

Multilingual

Multilingual

100% Multilingual including administrative interface

Has a French Canadian language pack, with many contributions from various government departments for common string elements, and has the ability to extend to over 200+ other languages...more than bilingual.

Migration

Migration

Robust scripted bulk-migration templates and cleansing

With fully automated bulk harvesting of CLF2, WET3 templates, DrupalWxT can instantly pull content from static sites, TeamSite, OpenText, IBM/Domino, and countless others

Content Staging

Content Staging

Full Content Staging and Deployment

With no code, you can run a full "staging > deploy" to a live production environment VIA an administrative interface with a bulk scheduled rules engine or on-demand page by page.

Workflow & Moderation

Workflow & Moderation

Workflow, Moderation & Auditing built-in

Includes a sophisticated workflow for both simple and complex processes. In-page audit tracking "who did what when" with versioning and ability to compare differences in revisions side by side, based on roles and permissions you have.

Forms

Forms

Forms Builder Interface

From simple user author-able forms for feedback, to comprehensive multifunction apps with integrated into legacy systems, DrupalWxT has a front-end to build it and manage it. (no code needed).

Layouts & Editors

Layouts & Editors

Content Layouts & Rich Text Editors

Offers content teams the option and ability to manage their layouts and page components with intuitive drag and drop panels UI. Includes an extensively featured rich text editor with user profile based restrictions, including media file handling and markup blacklists and whitelists... works with most rich text editors.

Search

Search

Search and Structure

Extends easily with SearchAPI and proven to work with Google Search and Solr for rich search application like facets and filters. Allows you to ingest external indexes from other systems and securely present metadata for a unified search results including full-text files.

There is more...

Responsive

WxT uses a mobile first approach based on the Omega theme, adapts to different screen sizes and device capabilities, has touchscreen support, is optimized for performance and is buliding support for device-based mobile applications.

Cross-Browser Compatibility

DrupalWxT has Cross-Browser Compatibility both in the front-end and the back-end, supporting Google Chrome, Mozilla Firefox, Safari, and IE8+.

Typography

DrupalWxT comes with plenty of default typography, giving you an awesome looking front and back end site.

Community

Each OpenSource project has thousands of Community based addons available for free, and with Source Code open. You often even have options for common requirements, so don’t reinvent the wheel.

Secure

Always a big one. There are 100’s of core developers from each projects that regularly monitor security risks.

Cost control

Stay out of the licensing gauntlet! Reduce integration, maintenance and operation cost... even upgrades.
